## 制作虚拟环境

制作虚拟环境

## 安装依赖

```bash
python.exe -m pip install -r requirements.txt
```

## 初始化数据库

```bash
python.exe .\manage.py makemigrations
python.exe .\manage.py migrate
```

## 创建超级用户

```bash
python.exe .\manage.py createsuperuser
```

## 启动服务器

```bash
python.exe .\mana.py runserver
```
